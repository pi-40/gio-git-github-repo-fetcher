## How To Use?
In order to have this program on your computer, you have to follow these steps:Step 1: Open the MSI and follow the instructions of the MSI (Microsoft Installer).Step 2: Once done, go into the directory where you put the program inside, and open the folder with the program name Gio-Git.Step 3: Open the bin directory, and follow it until you hit the last directory.Step 4: Once you hit the final directory, open the .exe file. Once you do, you should see a terminal-like window pop up. If you do, you can follow until Step 5. If not, then stay. If it closes immediately, you need to install .NET 10.0 on an x64 machine. If it still does not work, then check your Windows version or contact Microsoft Help.Step 5: In order to download a package inside a repo, you have to run: 
```
gio-git https://github.com/owner/repo
```
Use code with caution.If it works, press
the Down Arrow on your keyboard to go to
the packages menu. Once you are at the
packages, press A to download Release 1 in
alphabetical order (e.g., Release 1: press
A, Release 2: press B). It should start downloading once you do. If not, then check your spelling.Step 6: Once it gets done downloading your package, go inside the directory that has the program inside, and your package should be there.
